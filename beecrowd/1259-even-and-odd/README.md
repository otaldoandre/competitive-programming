# 1259 - Even and Odd

## Problem
Given an input of natural number values in sequence, output the values sorted in a specific order: the even numbers in ascending order followed by the odd numbers in descending order.

## Approach
Used two multiset: one for odd numbers (descending order) and one for even numbers (ascending order). 

## Key Concept
Multiset (automatic sorting with duplicates)

## Difficulties
None.

## What I Learned
Learned that the multiset data structure exists. Learned that multiset allows automatic sorting like a set, but also supports duplicate elements, which is useful in this problem.
