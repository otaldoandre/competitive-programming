# 2729 - Shopping List

## Problem
Given a list of items with possible repetitions, output them sorted without any duplicates.

## Approach
Used two vectors to solve: one to store all elements and one to store all elements with no duplicates. Then, sorted the second vector.

## Key Concept
Sorting / Duplicate removal.
